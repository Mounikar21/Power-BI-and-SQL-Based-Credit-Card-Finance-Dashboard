#  Credit Card Financial Report Dashboard

#  Project Description

This project focuses on building an advanced, weekly-updated dashboard to monitor and analyze credit card operations. The dashboard provides real-time visibility into key performance indicators (KPIs), enabling stakeholders to track trends, evaluate card usage behavior, and support data-driven decision-making. The goal is to streamline credit card management processes and drive strategic improvements through powerful visual insights.

# Tech Stack Used

**Power BI**
– for dashboard design and data visualization

**PostgreSQL & pgAdmin4**
– for database management and SQL-based data extraction

#  Skills showcased in this Project

- Data Extraction
- Data Cleaning
- Data Transformation
- Data Visualization


#  Project Approach


- Imported raw transaction and customer data into PostgreSQL database.

- Established a connection from Power BI to the SQL database.

- Cleaned, shaped, and transformed data within Power BI using Power Query.

- Created calculated metrics and KPIs using DAX to support business logic.

- Designed interactive dashboards with slicers and filters for user-driven exploration.

- Delivered analytical insights and visual reports tailored for stakeholder needs.
